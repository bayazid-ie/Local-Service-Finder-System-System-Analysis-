# Local Service Finder System - System Analysis Project


A web application that connects customers with local service providers.


## Features
- Search for Local Services - Plumbers, Electricians, Cleaners, etc.
- User & Vendor Registration - Separate registration systems
- Service Booking System - Book appointments with vendors
- Booking Management - View and cancel bookings
- Rating & Review System - Rate completed services
- Contact System - Customer support contact form


## Technology Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Icons: Font Awesome


## Project Structure
- MVC-like Architecture - Organized code structure
- Database-Driven - Dynamic content from MySQL
